Spellbook released seven updates in May, most notably launching a Google Docs add-in that allows users to run Reviews, Playbooks, and Ask features directly within Google Docs. The company introduced 12 new lawyer-vetted Starter Playbooks covering common contract types, and expanded Associate functionality with team collaboration features, improved file processing, natural language workflow building, mobile support, and expanded file type compatibility including email and image formats.
